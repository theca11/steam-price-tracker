Steam Price Tracker
About

Steam Price Tracker plugin displays the current price of any Steam app.
Keep track of discounts and don't miss out on a bargain!

▸ Automatically updates info every hour.
▸ Press key to go to the Steam Store app page.
▸ Long press key to force an update of all visible apps.

Troubleshooting

The plugin can't find an app.

Make sure you're writing the name of the app as it appears on the Steam Store. Partial matches will fail. If it still doesn't find it, use the app ID number instead of the name - it's part of the store page URL:

The displayed price is incorrect and/or is in the wrong currency.

By default prices are shown based on the store region detected by Steam. If this detection is wrong or you want to display prices of a different region, change the Store Country setting. This setting is shared among all keys, you only need to configure it once.
